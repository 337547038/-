Css Sprites

复制以下css样式到项目样式表

[class*="sprites-"]{background-image: url(../images/sprites.png); display: inline-block; vertical-align: middle}{{demoStyle}}

共 0 个图片

使用说明：

1、将所有需要合并的图片放在目录./src/sprites/icons下；

2、运行命令guilin sprites即可将所有指定的图片合并为sprites.png；新增图片时执行命令即可；

3、仅支持jpg和png两种格式，并且不支持中文命名；

4、如需重新生成，请手动删除文件./src/sprites/icons.json后重执行合并命令，重新生成后图片位置会发生变化，请慎重

5、将红色样式复制到项目样式，再显示图标位置添加对应class
